Type: stringFormat: date-time

The date and time when the image was captured. This is to be specified in UTC and conform to ISO 8601.

Must match regular expression: ^[0-9]{4}-(0[1-9]|1[0-2])-(0[1-9]|[12][0-9]|3[01])T([01][0-9]|2[0-3]):[0-5][0-9]:[0-5][0-9](\.[0-9]+)?(Z|[+-]([01][0-9]|2[0-3]):[0-5][0-9])?$
